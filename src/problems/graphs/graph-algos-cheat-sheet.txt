Single Source Shortest Path (for unweighted edges): BFS
Single Source Shortest Path for weighted edges (positive weights only): Dijkstra's (shortest path tree)
Single Source Shortest Path for weighted edges (positive and negative weights): Bellman Ford, SPFA
All source Shortest Path: Floyd Warshall
Minimum Spanning Tree: Kruskal's Algorithm, Prim's Algorithm
