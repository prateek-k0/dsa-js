Single Source Shortest Path (for unweighted edges): BFS (or Bi-BFS if start and end is known)
Single Source Shortest Path for weighted edges (+ve weights only): Dijkstra's (shortest path tree)
Single Source Shortest Path for weighted edges (+ve and -ve weights): Bellman Ford, SPFA
All source Shortest Path: Floyd Warshall
Minimum Spanning Tree: Kruskal's Algorithm, Prim's Algorithm
